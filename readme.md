## Сборка

Минимальная версия библиотеки Qt5.12.

Открыть battleRobot.pro в QtCreator и собрать используя имеющиеся комплекты.

или

Открыть CMakeLists.txt в QtCreator и собрать используя имеющиеся комплекты.
